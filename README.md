### 기능

- Create
    - 회원가입
    - 이용내역(마이페이지) 추가
- Read
    - 이용 가능한 자전거 조회
    - 이용내역(마이페이지) 조회
- Update
    - 대여/반납에 따른 자전거 상태 변경(1: 대여 가능 /0: 대여 중)
    - 대여/반납 시간 등록
- Delete
    - 회원탈퇴
